### Description
---
Format specifiers follow the format `%(char)` and are used to specify the type of data to be printed or scanned when using functions like `printf` and `scanf`. 

### Key Specifiers
---
- [[%p]]
- [[%d OR %i]] 
- [[%f]]
- [[%c]]
- [[%s]]
- [[%lf]]
- [[%u]]
- [[%x OR %X]]
- [[%o]]
- [[%ld]]
- [[%lld]]
